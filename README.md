# Rainfall Prediction Web Application
## Description 
This web application predicts rainfall using historical weather data and machine learning techniues.
It includes user authentication (login and registration) and display prediction results through a web interface.

## Objectives 
-Predict rainfall based on input data
-Provide user login and registration 
-Display prediction results on a web page 
-Support agriculture and analysis

## Technologies used 
-Python
-Flask
-Pandas 
-Numpay
-Matplotlib 
-Machine Learning 

#Project Structure 
-app.py - main Flask appication 
-templates/ - HTML files (login,register,dashbord)
-static/ - CSS, JS, images 
-model/ - dataset files 
-README.MD - project documentaion
